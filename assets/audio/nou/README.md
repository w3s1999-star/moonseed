# Nou Audio — Squeak Placeholder

Place a short (≈80–120 ms) soft squeak WAV file here named:

    nou_squeak.wav

Requirements:
- Mono, 44100 Hz
- Very short attack, soft sine-ish tone (think Animal Crossing blip)
- Roughly C5 or D5 pitch so pitch_scale variation (0.88–1.12) sounds natural

This directory previously contained audio files for the NouMascot mascot system.
The mascot system has been removed from the project.
